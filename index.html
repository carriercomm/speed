Speed

Fault tolerant distributed in memory file cache based on Membase

Download this project as a .zip file Download this project as a tar.gz file

This is a library which provides completely asynchronous, fault tolerant distributed in memory file cache.

Primary expectation

  1. High concurrent read of same file with very low latency from multiple clients & high through put.
  2. High concurrent read of different file with very low latency from multiple clients & high through put.
  3. Lockless read/write.
  4. Highly available and fault tolerant.

Secondary expectation

  1. Random seeks
  2. Random writes to a file
